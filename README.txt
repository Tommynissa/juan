Livret d'accueil — Appartement Juan-les-Pins
=============================================

Contenu du zip :

- index.html
- style.css
- script.js
- images/placeholder-photo.jpg (image d'exemple)
- README.txt (ce fichier d'instructions)

Instructions rapides :
1. Dézippez et testez en local : ouvrez index.html dans un navigateur.
2. Remplacez les valeurs suivantes avant publication :
   - Wi‑Fi (dans index.html) pour mettre le vrai mot de passe.
   - Clé Google Maps Embed API : remplacez VOTRE_CLE_API dans le iframe de index.html.
   - Clé OpenWeatherMap : remplacez VOTRE_API_METEO dans script.js.
3. Hébergement :
   - Netlify : glissez le dossier dans Netlify Drop ou connectez un dépôt GitHub.
   - GitHub Pages : poussez le repo et activez GitHub Pages (branch: main ou gh-pages).
4. QR code :
   - Une fois l'URL publique disponible, génère un QR code (j'utilise un service externe ou je peux le générer pour toi si tu me donnes l'URL).
5. Personnalisation :
   - Tu peux remplacer images/placeholder-photo.jpg par une photo de ton appartement (taille recommandée : 1200×800).

Besoin d'aide pour déployer ? Donne-moi l'URL une fois en ligne et je te génère un QR code prêt à imprimer.
